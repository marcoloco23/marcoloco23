## How I build

```python
def build(problem):
    """The loop I actually run."""
    understand(problem)          # slow down here — most bugs start as bad assumptions
    solution = simplest_thing_that_could_work(problem)

    while not solution.ships():
        solution = refine(solution)          # correctness first
        solution = make_it_fast(solution)    # then, and only then, speed
        assert solution.is_boring()          # clever is a liability at 3am

    return solution
```
